# 🚀 Termux Power Setup

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/bhu01hackerspace/termux-power-setup)
[![Termux](https://img.shields.io/badge/Termux-Android-orange.svg)](https://termux.com/)
[![Bash](https://img.shields.io/badge/Shell-Bash-green.svg)](https://www.gnu.org/software/bash/)

Script completo para configurar o Termux com as melhores ferramentas para desenvolvimento, redes e produtividade.

## 📋 Funcionalidades

### 🐍 Linguagens de Programação
- Python (+ pip, requests, beautifulsoup4, numpy, pandas, colorama)
- Node.js
- Ruby
- PHP
- C/C++ (Clang)
- Java (OpenJDK-17)

### 🌐 Ferramentas de Rede
- Nmap (scanner de portas)
- OpenSSH (acesso remoto)
- Netcat (debug de conexões)
- Wget/Curl (downloads e requisições)
- Tor (navegação anônima)
- Traceroute (rota de pacotes)
- WhoIs (consulta de domínios)
- Hydra (brute-force - opcional)
- Metasploit (framework de segurança - opcional)

### 🔧 Utilitários
- Htop (monitor de processos)
- Git (controle de versão)
- Nano/Vim (editores de texto)
- Tmux (multiplexador de terminal)
- Tree (estrutura de pastas)
- Ranger (gerenciador de arquivos)
- FFmpeg (processador de mídia)
- Neofetch (info do sistema)
- Cmatrix (efeito Matrix)
- Zsh + Oh My Zsh (shell turbinado)

### ⚡ Configurações
- Aliases personalizados (ll, update, ipinfo, weather, etc.)
- Estrutura de diretórios (projetos, scripts, tools, backups)
- Sistema de log completo

## 🚀 Como usar

### Instalação rápida
```bash
git clone https://github.com/bhu01hackerspace/termux-power-setup.git
cd termux-power-setup
chmod +x setup-termux.sh
./setup-termux.sh
